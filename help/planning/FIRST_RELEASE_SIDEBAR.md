## Why This Scope

- It gives new users a fast path to first success.
- It provides enough workflow-authoring depth to create real pipelines.
- It introduces validation, persistence, and observability early.
- It adds a small reference section without overwhelming the launch.
- It reserves advanced extension and deep runtime topics for later phases.
